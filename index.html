Sistema de Ensaque — Configuracao
Conecte ao Firebase para comecar. Leva 5 minutos.
1
Criar projeto no Firebase
Acesse console.firebase.google.com, clique em Adicionar projeto, dê um nome (ex: sistensaque) e crie o projeto.
2
Ativar o Realtime Database
No menu lateral, clique em Build → Realtime Database → Criar banco de dados. Escolha a localização us-central1 e inicie em modo de teste (você vai proteger depois).
3
Pegar as configurações do projeto
Clique na engrenagem ⚙ → Configurações do projeto → Seus aplicativos → Adicionar app → Web (</>). Registre o app e copie o objeto firebaseConfig que aparecer.
4
Cole as configurações abaixo
Preencha todos os campos.
